# maaltafels
tegen de tijd maaltafels oefenen

python 2 script with TKinter GUI
doel: kinderen maaltafels laten oefenen, met registratie van de tijd. 
doelpubliek: kinderen van 6 tot 9 jaar.

werking programma:
Om de oefeningen te genereren, druk de groene knop in. Een nieuw scherm opent. Daarin wordt de naam van het kind gevraagd, alsook zijn nummer binnen de klas. 
De grijze knop start de test. 

De test omvat 50 vermenigvuldigingen en 50 delingen. Beide zijn gescheiden van elkaar, zodat er een overzichtelijk schema ontstaat. 
Er kan door de antwoordvakjes gegaan worden door middel van de tab toets (terug gaan via shift + tab). Dit gaat sneller dan klikken, de nul verwijderen en de uitkomst in te vullen.

Op het einde van de test wordt het resultaat weggeschreven naar een .txt-file. Deze file bevat de naam van de deelnemer, de datum van de test, het resultaat op 100 en de tijd die de deelnemer nodig had. Deze file kan dienen om progressie in te schatten.
